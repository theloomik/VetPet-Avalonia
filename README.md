# VetPet-Avalonia

Кросплатформний десктопний застосунок для адміністрування ветеринарної клініки, виконаний у межах практики.
Проєкт побудований на **Avalonia UI + .NET 8** з архітектурою **MVVM** та інтеграцією з **MySQL (EF Core)**.

## Мета проєкту

Надати зручний інтерфейс для щоденної роботи адміністраторів клініки:
- ведення клієнтів і їхніх тварин;
- планування записів на прийом;
- облік працівників;
- облік постачальників;
- контроль складських залишків.

## Основний функціонал

У застосунку реалізовані окремі модулі (вкладки):
- **Клієнти** — список клієнтів, перегляд і перехід до деталей клієнта;
- **Записи** — робота з прийомами;
- **Працівники** — облік персоналу;
- **Провайдери** — робота з постачальниками;
- **Склад** — облік медикаментів і залишків.

> Навігація між модулями реалізована через `MainWindowViewModel` і динамічну підстановку відповідних `ViewModel`.

## Технології

- **.NET 8**
- **Avalonia 11**
- **CommunityToolkit.Mvvm**
- **Entity Framework Core 9**
- **Pomelo.EntityFrameworkCore.MySql**
- **MySqlConnector**

## Примітки

- Проєкт створено як навчально-практичну роботу.
- Деякі дані залежать від поточного стану локальної бази `vetpet`.
- За потреби можна додати SQL-скрипт ініціалізації БД у майбутніх ітераціях.
